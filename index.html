🔥 Connecter Firebase
Firebase permet la synchronisation en temps réel entre votre admin, les restaurants et les livreurs. C'est gratuit.
1
Allez sur console.firebase.google.com → Créez un projet → Nommez-le troprapide
2
Dans votre projet → Realtime Database → Créer une base de données → Mode test → Copier l'URL
3
Dans Paramètres du projet → Vos applications → Icône Web → Enregistrer → Copier les infos ci-dessous
Ces informations sont sauvegardées localement sur votre navigateur.
Vous n'aurez à les saisir qu'une seule fois.
